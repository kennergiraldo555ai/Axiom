# AXIOM TASK

## Objective

Describe the requested implementation.

---

## Before Coding

Understand the problem.

Read the affected files.

Find the root cause.

Do not code yet.

Explain the solution.

Only then implement.

---

## Constraints

Work only in localhost.

Never deploy.

Never commit.

Never push.

Never merge.

Never modify Vercel.

Never modify GitHub.

Never use browser automation.

---

## Mandatory Validation

Run:

pnpm lint

pnpm typecheck

pnpm build

If one fails:

Stop.

Fix.

Repeat.

---

## Deliverables

Provide:

### Root Cause

Explain the actual issue.

---

### Files Modified

List every modified file.

---

### Why the Solution Works

Explain technically.

---

### Risks

List any remaining risks.

---

### Validation

Report:

✅ lint

✅ typecheck

✅ build

---

### Manual Testing

Explain exactly how the user should verify the feature in localhost.

Never claim functionality is verified.

Only the user can verify behavior.

---

### Technical Debt

Describe any pending improvements that should be addressed in future sprints.

---

### Recommended Next Task

Suggest only one logical next step.

Do not implement it.

Wait for approval.
